## Usage

Clone the repository:
```bash
  git clone https://github.com/sirireddy25/invigilation_duty_booking.git
```
Install dependencies:
```bash
  pip install -r requirements.txt
```
Run migrations:
```bash
  python manage.py migrate
```
Start the server:
```bash
  python manage.py runserver
```
